How to Use the Foursquare Leaderboard API

Back Home

The Foursquare Leaderboard API allows you to access a list of users ranked based on their performance. This guide will walk you through the steps to extract information from the API and retrieve specific details about individual players.

Step 1: Accessing the User List

To get a list of all users ranked from top to bottom, you need to use the following fetch request:

fetch('https://foursquare-leaderboard-default-rtdb.firebaseio.com/leaderboard.json', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json',
  },
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error fetching the leaderboard:', error));

When you execute this fetch request, you will receive a list of players. The players are ordered by their rank, with the top players at the top of the list and the bottom players at the bottom. For example:

      {
        "-NvvOLwaGy5BEHZf4zpk": {
            "downvotes": 3,
            "name": "Player1",
            "rank": 1,
            "upvotes": 8
        },
        "-NxwMFGwUhSO6vdpycZd": {
            "downvotes": 2,
            "name": "Player2",
            "rank": 2,
            "upvotes": 7
        },
        "-NxwMFGwUhSO6vdpycZe": {
            "downvotes": 0,
            "name": "Player3",
            "rank": 3,
            "upvotes": 7
        }
    }

In the example above, Player1 has 3 Downvotes, 8 Upvotes, and the rank of 1. Player2 has 2 Downvotes, 7 Upvotes, the rank of 2... etc and so own down the list for each player.
